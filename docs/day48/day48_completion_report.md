## 10.1 Prohibited public content

Do not publish:

* Macy’s internal source code

* Internal architecture diagrams
* Company endpoints
* Credentials
* API keys
* HMAC secrets
* Certificates
* Production tokens
* Customer records
* Employee records
* Proprietary database schemas
* Private Confluence content
* Internal email or chat content

## 10.2 Recommended repository scan

Run a manual review for sensitive terms:

```powershell
Get-ChildItem -Path . -Recurse -File |
  Select-String -Pattern `
    "macys|macy's|bloomingdales|zensar|ultra global|apikey|secret|password|certificate|private key"
```

Every match must be reviewed manually because some generic terms may be valid in test or documentation contexts.
